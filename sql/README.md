# SQL Scripts

This directory contains all SQL scripts for database setup, migrations, and utilities.

## Structure

- **schema/** - Database schema definitions
- **migrations/** - Database migration scripts
- **seeds/** - Seed data scripts
- **queries/** - Common query utilities

## Usage

Run SQL scripts against your Supabase PostgreSQL database using:
- Supabase SQL Editor
- psql command line
- Database migration tools

## Naming Convention

- Schema files: `schema_<table_name>.sql`
- Migrations: `migration_<timestamp>_<description>.sql`
- Seeds: `seed_<table_name>.sql`
- Queries: `query_<description>.sql`
